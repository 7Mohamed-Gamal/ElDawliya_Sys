# دليل البدء السريع - نظام الدولية

## 🚀 ابدأ في 5 دقائق

هذا الدليل سيساعدك على البدء في استخدام نظام الدولية بسرعة وسهولة.

## الخطوة 1: تسجيل الدخول

1. افتح متصفح الويب (Chrome, Firefox, Safari, Edge)
2. اذهب إلى رابط النظام: `http://your-server-address:8000`
3. أدخل بيانات تسجيل الدخول:
   - **اسم المستخدم**: الذي حصلت عليه من المدير
   - **كلمة المرور**: كلمة المرور المؤقتة
4. اضغط **تسجيل الدخول**

> 💡 **نصيحة**: إذا كانت هذه المرة الأولى، ستحتاج لتغيير كلمة المرور

## الخطوة 2: تغيير كلمة المرور (أول مرة)

1. بعد تسجيل الدخول، اضغط على اسمك في الزاوية اليمنى العلوية
2. اختر **الملف الشخصي**
3. اضغط **تغيير كلمة المرور**
4. أدخل:
   - كلمة المرور الحالية (المؤقتة)
   - كلمة المرور الجديدة (قوية ومعقدة)
   - تأكيد كلمة المرور الجديدة
5. اضغط **حفظ**

## الخطوة 3: استكشاف الواجهة الرئيسية

### الشريط العلوي
- **الشعار**: العودة للصفحة الرئيسية
- **البحث**: البحث السريع في جميع البيانات
- **الإشعارات**: 🔔 آخر التحديثات والتنبيهات
- **الملف الشخصي**: إعداداتك الشخصية

### القائمة الجانبية
- **لوحة المعلومات**: الإحصائيات والملخصات
- **الموارد البشرية**: إدارة الموظفين والحضور
- **المخزون**: إدارة المنتجات والمواد
- **المشاريع**: إدارة المهام والاجتماعات
- **المشتريات**: أوامر الشراء والموردين
- **التقارير**: جميع التقارير والإحصائيات

## الخطوة 4: المهام الأساسية الأولى

### إذا كنت مدير موارد بشرية:

#### إضافة موظف جديد (دقيقتان)
1. **الموارد البشرية** → **الموظفين**
2. **إضافة موظف جديد**
3. املأ البيانات الأساسية:
   ```
   رقم الموظف: EMP001
   الاسم الأول: أحمد
   الاسم الأخير: محمد
   البريد الإلكتروني: ahmed@company.com
   رقم الهاتف: 01234567890
   ```
4. اختر **القسم** و **المنصب**
5. **حفظ**

#### تسجيل حضور (30 ثانية)
1. **الموارد البشرية** → **الحضور والانصراف**
2. **تسجيل حضور جديد**
3. اختر **الموظف** من القائمة
4. الوقت سيسجل تلقائياً
5. **حفظ**

### إذا كنت مدير مخزون:

#### إضافة منتج جديد (دقيقتان)
1. **المخزون** → **المنتجات**
2. **إضافة منتج جديد**
3. املأ البيانات:
   ```
   كود المنتج: PRD001
   اسم المنتج: لابتوب ديل
   الفئة: أجهزة كمبيوتر
   سعر الشراء: 15000
   سعر البيع: 18000
   ```
4. **حفظ**

#### تسجيل حركة مخزون (دقيقة واحدة)
1. **المخزون** → **حركات المخزون**
2. **حركة وارد**
3. اختر **المنتج** والكمية
4. أدخل **رقم الفاتورة**
5. **حفظ**

### إذا كنت مدير مشاريع:

#### إنشاء مشروع جديد (3 دقائق)
1. **المشاريع** → **المشاريع**
2. **مشروع جديد**
3. املأ البيانات:
   ```
   اسم المشروع: تطوير موقع الشركة
   الوصف: إنشاء موقع إلكتروني جديد
   تاريخ البداية: اليوم
   تاريخ النهاية: بعد 3 أشهر
   ```
4. اختر **مدير المشروع**
5. **حفظ**

#### إضافة مهمة (دقيقة واحدة)
1. من داخل المشروع → **إضافة مهمة**
2. املأ:
   ```
   عنوان المهمة: تصميم الواجهة الرئيسية
   المسؤول: اختر من القائمة
   تاريخ الانتهاء: خلال أسبوع
   الأولوية: عالية
   ```
3. **حفظ**

## الخطوة 5: استكشاف التقارير

### عرض لوحة المعلومات
1. اضغط على **لوحة المعلومات** في القائمة الجانبية
2. ستجد:
   - إحصائيات الموظفين
   - حالة المخزون
   - تقدم المشاريع
   - أوامر الشراء المعلقة

### إنشاء تقرير سريع
1. **التقارير** → اختر نوع التقرير
2. حدد **الفترة الزمنية**
3. اضغط **إنشاء التقرير**
4. يمكنك **تصدير** التقرير بصيغة Excel أو PDF

## الخطوة 6: تخصيص الإعدادات

### إعدادات الملف الشخصي
1. اضغط على **اسمك** → **الملف الشخصي**
2. حدث:
   - **الصورة الشخصية**
   - **رقم الهاتف**
   - **المنطقة الزمنية**
   - **لغة الواجهة**

### إعدادات الإشعارات
1. من الملف الشخصي → **إعدادات الإشعارات**
2. اختر أنواع الإشعارات التي تريد استلامها:
   - ✅ إشعارات البريد الإلكتروني
   - ✅ الإشعارات الفورية
   - ⬜ إشعارات الهاتف

## 🆘 إذا واجهت مشكلة

### مشاكل تسجيل الدخول
- تأكد من صحة اسم المستخدم وكلمة المرور
- تأكد من اتصالك بالإنترنت
- جرب متصفح آخر
- اتصل بمدير النظام

### مشاكل في الأداء
- أغلق التبويبات غير المستخدمة
- امسح ذاكرة التخزين المؤقت للمتصفح
- تأكد من سرعة الإنترنت
- جرب في وقت أقل ازدحاماً

### لا تجد ميزة معينة
- استخدم **البحث** في الشريط العلوي
- راجع **دليل المستخدم** الكامل
- اضغط على **المساعدة** في أي صفحة
- اتصل بالدعم الفني
