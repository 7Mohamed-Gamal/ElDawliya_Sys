## 🛠️ كيفية الاستخدام

### 1. إعداد البيئة التطويرية
```bash
# تشغيل سكريبت الإعداد
python setup_development.py

# أو يدوياً
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
```

### 2. إنشاء الهجرات المحسنة
```bash
python manage.py create_enhanced_migrations
```

### 3. إنشاء بيانات تجريبية
```bash
python manage.py create_demo_data --companies 3 --employees 50
```

### 4. تشغيل النظام
```bash
# تشغيل الخادم
python manage.py runserver

# تشغيل Celery (في terminal منفصل)
celery -A ElDawliya_sys worker -l info

# تشغيل Celery Beat (في terminal منفصل)
celery -A ElDawliya_sys beat -l info
```
