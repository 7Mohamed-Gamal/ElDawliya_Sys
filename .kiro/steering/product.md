# ElDawliya System (نظام الدولية)

ElDawliya System is a comprehensive enterprise management platform built for Arabic-speaking organizations. It's an integrated business management system that handles HR, inventory, meetings, tasks, and purchasing operations.

## Core Features

- **HR Management**: Employee records, attendance, payroll, leave management
- **Inventory Management**: Stock tracking, supplier management, automated purchase orders
- **Task Management**: Personal and team task tracking with progress monitoring
- **Meeting Management**: Scheduling, participant management, minutes recording
- **Purchase Orders**: Procurement workflow with approval processes
- **AI Integration**: Google Gemini AI for data analysis and intelligent conversations
- **REST API**: Comprehensive API with Swagger documentation for all modules

## Key Characteristics

- **RTL Support**: Built for Arabic language with right-to-left text direction
- **Multi-database**: Supports SQL Server with fallback mechanisms
- **Role-based Access**: Hierarchical permissions system (department > module > permission)
- **Audit Trail**: Comprehensive logging of all user actions and data changes
- **Notification System**: Real-time notifications across all modules
- **Modern UI**: Bootstrap 5 with Arabic font support (Cairo font)

## Target Users

Enterprise organizations requiring integrated management of HR, inventory, and operational processes with Arabic language support.
