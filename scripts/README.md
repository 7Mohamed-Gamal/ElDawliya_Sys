_# Scripts Directory

This directory contains utility scripts for system maintenance and data migration.

## One-Time Migration Scripts (can be removed after use)
- `comprehensive_migration.py` - Full data migration script
- `consolidate_apps.py` - App consolidation helper
- `data_migration.py` - Data migration between databases
- `final_migration_script.py` - Final migration cleanup
- `fix_critical_issues.py` - One-time critical fixes
- `fix_migrations.py` - Migration file fixes
- `migrate_configuration.py` - Configuration migration
- `refactor_codebase.py` - Codebase refactoring helper
- `update_app_configs.py` - App configuration updates

## Recurring/Utility Scripts
- `analyze_system_status.py` - System health analysis
- `code_quality_review.py` - Code quality checker
- `create_backup.py` - Database backup creation
- `test_backup_structure.py` - Backup validation
- `test_configuration.py` - Configuration testing
- `test_inventory_models_syntax.py` - Inventory model validation
- `test_migrations.py` - Migration testing
- `test_project_models_syntax.py` - Project model validation
- `validate_backup_system.py` - Backup system validation
- `validate_inventory_models.py` - Inventory model validation
- `validate_migrations.py` - Migration validation
- `validate_project_models.py` - Project model validation
