# Scripts Directory

This directory contains utility scripts for the ElDawliya System.

## Files

- `install_packages.ps1` - PowerShell script for package installation
- `run_checks.bat` - Batch script for running system checks

## Usage

Run scripts from the project root directory:

```bash
# PowerShell scripts
powershell -ExecutionPolicy Bypass -File scripts/install_packages.ps1

# Batch scripts  
scripts/run_checks.bat
```
