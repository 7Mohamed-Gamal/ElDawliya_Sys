# نظام الشركة الدولية إنترناشونال (ElDawliya System)

![شعار الشركة الدولية](static/images/logo.png)

نظام متكامل لإدارة الشركة الدولية إنترناشونال، يشمل إدارة الموارد البشرية، المخزون، الاجتماعات، المهام، وطلبات الشراء.

## 🌟 المميزات الرئيسية

### 🧑‍💼 نظام الموارد البشرية (HR)
- إدارة شاملة لبيانات الموظفين
- إدارة الأقسام والوظائف
- نظام الرواتب والمستحقات
- إدارة الحضور والانصراف
- إدارة الإجازات والتقييمات
- الهيكل التنظيمي للشركة
- تقارير وإحصائيات متنوعة

### 📦 نظام المخزون (Inventory)
- إدارة قطع الغيار والمنتجات
- إدارة المخزون والجرد
- إدارة الموردين والعملاء
- إدارة الفواتير والمشتريات
- تقارير المخزون والتنبيهات

### 🤝 نظام الاجتماعات (Meetings)
- جدولة وإدارة الاجتماعات
- إدارة الحضور
- ربط المهام بالاجتماعات
- متابعة قرارات الاجتماعات

### ✅ نظام المهام (Tasks)
- إنشاء وتعيين المهام
- متابعة حالة المهام
- تقارير إنجاز المهام
- إشعارات المهام المتأخرة

### 🛒 نظام طلبات الشراء (Purchase Orders)
- إدارة طلبات الشراء
- متابعة حالة الطلبات
- ربط الطلبات بالمخزون
- الموافقات والصلاحيات

### 👮‍♂️ نظام الصلاحيات (Permissions)
- إدارة مستخدمي النظام
- تحديد صلاحيات الوصول
- مجموعات المستخدمين
- تقييد الوصول للأقسام والوظائف

## 🛠️ التقنيات المستخدمة

- **لغة البرمجة**: Python 3.11+
- **إطار العمل**: Django 5.0
- **قاعدة البيانات**: SQL Server / SQLite
- **واجهة المستخدم**: HTML5, CSS3, JavaScript, Bootstrap 5
- **مكتبات إضافية**: 
  - django-widget-tweaks
  - mssql-django
  - xlwt (لتصدير التقارير)

## 📋 متطلبات النظام

- Python 3.11 أو أحدث
- SQL Server 2019 أو أحدث (أو SQLite للتطوير)
- ODBC Driver 17 for SQL Server
- متصفح حديث (Chrome, Firefox, Edge)

## ⚙️ التثبيت والإعداد

### 1. استنساخ المشروع

```bash
git clone https://github.com/yourusername/ElDawliya_System.git
cd ElDawliya_System
```

### 2. إنشاء بيئة افتراضية وتثبيت المتطلبات

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### 3. إعداد قاعدة البيانات

#### استخدام SQL Server:
- تأكد من تثبيت SQL Server وODBC Driver 17
- قم بإنشاء قاعدة بيانات باسم `El_Dawliya_International`
- عدّل إعدادات الاتصال في ملف `settings.py`

#### استخدام SQLite (للتطوير):
- قم بتعديل إعدادات قاعدة البيانات في `settings.py` لاستخدام SQLite

### 4. تطبيق الترحيلات وإنشاء مستخدم مدير

```bash
python manage.py migrate
python manage.py createsuperuser
```

### 5. تشغيل الخادم المحلي

```bash
python manage.py runserver
```
