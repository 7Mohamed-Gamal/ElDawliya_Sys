# تقرير اختبارات الأداء والتحميل
# Performance and Load Testing Report

## نظرة عامة

تم تطوير مجموعة شاملة من اختبارات الأداء والتحميل لنظام الدولية تشمل:

- **اختبارات أوقات الاستجابة**: قياس أداء الصفحات والـ API
- **اختبارات التحميل المتزامن**: محاكاة مستخدمين متعددين
- **اختبارات استهلاك الذاكرة**: مراقبة استهلاك الموارد
- **اختبارات التحسين**: تحليل وتحسين الأداء

## الملفات المطورة

### 1. ملفات الاختبار الأساسية

#### `test_page_response_times.py`
- **الغرض**: قياس أوقات الاستجابة للصفحات الرئيسية
- **الاختبارات المتضمنة**:
  - اختبار الصفحة الرئيسية (< 500ms)
  - اختبار صفحات الموارد البشرية (< 1000ms)
  - اختبار صفحات المخزون (< 1000ms)
  - اختبار API endpoints (< 300ms)
  - تحليل أداء استعلامات قاعدة البيانات
  - اختبار أداء التخزين المؤقت

#### `test_load_testing.py`
- **الغرض**: اختبار التحميل المتزامن للمستخدمين المتعددين
- **الاختبارات المتضمنة**:
  - تحميل خفيف (5 مستخدمين، معدل نجاح ≥ 95%)
  - تحميل متوسط (10 مستخدمين، معدل نجاح ≥ 90%)
  - تحميل ثقيل (20 مستخدم، معدل نجاح ≥ 80%)
  - اختبار العمليات المتزامنة على قاعدة البيانات

#### `test_memory_usage.py`
- **الغرض**: مراقبة استهلاك الذاكرة والموارد
- **الاختبارات المتضمنة**:
  - مراقبة استهلاك الذاكرة للصفحات
  - كشف تسريب الذاكرة في الطلبات المتكررة
  - اختبار استهلاك الذاكرة مع الطلبات المتزامنة
  - مراقبة استهلاك المعالج وعمليات القرص

#### `test_optimization.py`
- **الغرض**: تحليل وتحسين الأداء
- **الاختبارات المتضمنة**:
  - تحليل وتحسين استعلامات قاعدة البيانات
  - كشف مشكلة N+1 في الاستعلامات
  - اختبار فعالية select_related/prefetch_related
  - تحليل أداء التخزين المؤقت
  - قياس تأثير التحسينات على الأداء

### 2. أدوات التشغيل والإدارة

#### `performance_test_runner.py`
- **الغرض**: مشغل اختبارات الأداء الشامل
- **الميزات**:
  - تشغيل جميع اختبارات الأداء تلقائياً
  - إنتاج تقارير شاملة بتنسيق JSON
  - تحليل النتائج وإنتاج التوصيات
  - حفظ النتائج مع timestamp

#### `run_performance_tests.py`
- **الغرض**: سكريبت تشغيل اختبارات الأداء من سطر الأوامر
- **الميزات**:
  - إعداد البيئة تلقائياً
  - تشغيل هجرات قاعدة البيانات
  - إنشاء بيانات اختبار
  - تنظيف البيانات بعد الاختبار
  - تقارير مفصلة عن حالة النظام

#### `simple_performance_test.py`
- **الغرض**: اختبار أداء بسيط ومستقل
- **الميزات**:
  - لا يعتمد على تكوين Django الكامل
  - اختبار موارد النظام الأساسية
  - اختبار أداء الذاكرة والمعالج
  - اختبار أداء الإدخال والإخراج
  - اختبار العمليات المتزامنة

### 3. ملفات التكوين والإعدادات

#### `performance_config.py`
- **الغرض**: إعدادات وثوابت اختبارات الأداء
- **المحتويات**:
  - معايير الأداء المستهدفة
  - إعدادات الاختبارات المختلفة
  - URLs للاختبار
  - إعدادات التقارير والتنبيهات
  - دوال المساعدة للتقييم

#### `requirements/performance.txt`
- **الغرض**: متطلبات اختبارات الأداء
- **المتطلبات المتضمنة**:
  - psutil: مراقبة النظام والعمليات
  - memory-profiler: تحليل استهلاك الذاكرة
  - django-silk: تحليل أداء Django
  - locust: اختبارات التحميل المتقدمة
  - pytest-benchmark: قياس الأداء
  - matplotlib, pandas: تحليل وتصور البيانات

## معايير الأداء المحددة

### أوقات الاستجابة المستهدفة
- **الصفحة الرئيسية**: < 500ms
- **صفحات القوائم**: < 1000ms
- **صفحات التفاصيل**: < 800ms
- **صفحات النماذج**: < 600ms
- **API endpoints**: < 300ms
- **صفحات البحث**: < 1200ms

### معايير التحميل المتزامن
- **تحميل خفيف (5 مستخدمين)**: معدل نجاح ≥ 95%
- **تحميل متوسط (10 مستخدمين)**: معدل نجاح ≥ 90%
- **تحميل ثقيل (20 مستخدم)**: معدل نجاح ≥ 80%
- **اختبار الضغط (50 مستخدم)**: معدل نجاح ≥ 70%

### معايير استهلاك الموارد
- **زيادة الذاكرة لكل صفحة**: < 50MB
- **زيادة الذاكرة الإجمالية للجلسة**: < 100MB
- **حد تسريب الذاكرة**: < 20MB
- **استهلاك المعالج الأقصى**: < 80%

### معايير قاعدة البيانات
- **أقصى عدد استعلامات لكل صفحة**: < 50
- **متوسط وقت الاستعلام**: < 10ms
- **حد الاستعلام البطيء**: < 100ms
- **أقصى عدد استعلامات بطيئة**: < 5

## كيفية الاستخدام

### تشغيل جميع الاختبارات
```bash
python run_performance_tests.py
```

### تشغيل اختبارات محددة
```bash
# اختبارات أوقات الاستجابة
python run_performance_tests.py --type response

# اختبارات التحميل
python run_performance_tests.py --type load

# اختبارات الذاكرة
python run_performance_tests.py --type memory

# اختبارات التحسين
python run_performance_tests.py --type optimization
```

### تشغيل الاختبار البسيط
```bash
python tests/performance/simple_performance_test.py
```

### تشغيل المشغل الشامل
```bash
python tests/performance/performance_test_runner.py
```
